# 贡献指南

感谢你对 AI-2048 的关注！以下是参与贡献的方式：

## 🐛 提交问题

1. 在 [Issues](../../issues) 页面搜索是否已有类似问题
2. 使用 Issue 模板描述问题（含复现步骤）

## 🔧 提交代码

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/xxx`
3. 提交代码：`git commit -m "feat: 添加xxx功能"`
4. 推送分支：`git push origin feature/xxx`
5. 提交 Pull Request

## 📝 提交规范

遵循约定式提交：
- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档
- `refactor:` 重构
- `test:` 测试
- `chore:` 其他

## ✅ 代码审查

所有 PR 会经过：
1. 自动化 Lint 检查
2. 单元测试
3. AI 代码审查
