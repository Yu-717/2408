# AI Generated Boundary Test Suggestions

> 说明：以下用例由 AI 测试生成流程产出，进入正式测试前需要人工确认。

## slideRow(row)

| 场景 | 输入 | 期望 |
|---|---|---|
| 相邻合并 | `[2,2,0,0]` | `[4,0,0,0]` |
| 隔空合并 | `[0,2,0,2]` | `[4,0,0,0]` |
| 连续三个相同值 | `[2,2,2,0]` | `[4,2,0,0]` |
| 四个相同值 | `[2,2,2,2]` | `[4,4,0,0]` |
| 全零边界 | `[0,0,0,0]` | `[0,0,0,0]` |

## checkLose()

- 满盘且横向、纵向都不能合并：返回 `true`。
- 满盘但存在横向相邻相同方块：返回 `false`。
- 满盘但存在纵向相邻相同方块：返回 `false`。
- 仍存在空格：返回 `false`。

## spawnTile()

- 棋盘存在空格：非零方块数量增加 1。
- 棋盘无空格：返回 `false` 且棋盘不变。

## undoMove()

- 有效移动后撤销：棋盘和分数恢复到移动前。
- 无快照时撤销：返回 `false`。

## 棋盘尺寸

- `setBoardSize(3)` 后棋盘为 3×3，初始非零方块为 2。
- `setBoardSize(5)` 后棋盘为 5×5，初始非零方块为 2。
