# 🎮 AI-2048

> 开源软件课程项目 — Harness驱动 + AI能力集成 + CI/CD质量管控

一个经典的2048数字合并游戏，采用现代化的CI/CD流水线和AI驱动的代码质量管理。

## ✨ 特性

- 🎯 经典2048玩法，键盘+触屏双支持
- 🤖 AI驱动的代码审查与质量门禁
- 🔄 Harness CI/CD 全自动流水线
- ✅ 自动化测试覆盖（单元测试 + 边界测试）
- 📊 代码覆盖率报告

## 🚀 快速开始

直接在浏览器中打开 `game2048/index.html` 即可游玩。

或使用本地服务器：

```bash
npx serve game2048
```

## 🎮 玩法

- 方向键 `↑ ↓ ← →` 或 `WASD` 移动方块
- 相同数字合并，目标到达 **2048**
- 移动端支持触屏滑动

## 🏗️ 项目结构

```
game2048/
├── index.html          # 游戏主文件（HTML+CSS+JS）
├── test/               # 测试用例
├── .github/            # GitHub Actions / Harness 配置
└── docs/               # 项目文档
```
